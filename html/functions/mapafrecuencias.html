Función: Mapa de frecuenciasVolver

Descripción

Devuelve un mapa de frecuencias de elementos de un vector, valores de diccionario, caracteres de cadena o dígitos de un entero

Cadena de entrada

mapafrec

Cadena de salida

mapafrec

Uso

mapafrec(<vector>|<diccionario>|<texto>|<entero>)

Ejemplos

mapafrec(5,2,'viviana','juan','pedro','juan',2,true,false,true,true,[],[])

Diccionario: { []: 2 'juan': 2 5: 1 verdadero: 3 'viviana': 1 'pedro': 1 falso: 1 2: 2 }

mapafrec(elegir(rango(1,7),1E6,true))

Diccionario: { 2: 166547 3: 166607 1: 166795 4: 166401 5: 166697 6: 166953 }

mapafrec('Lorem ipsum dolor sit amet, consectetur adipiscing elit')

Diccionario: { 'a': 2 'c': 3 'd': 2 'e': 5 'g': 1 'i': 6 'L': 1 'l': 2 ',': 1 'm': 3 'n': 2 'o': 4 'p': 2 'r': 3 's': 4 't': 5 'u': 2 ' ': 7 }

mapafrec(reemplazar('Lorem ipsum dolor sit amet, consectetur adipiscing elit','\W'))

Diccionario: { 'a': 2 'c': 3 'd': 2 'e': 5 'g': 1 'i': 6 'L': 1 'l': 2 'm': 3 'n': 2 'o': 4 'p': 2 'r': 3 's': 4 't': 5 'u': 2 }

mapafrec(1000b!)

Diccionario: { 0: 472 1: 239 2: 248 3: 216 4: 229 5: 213 6: 231 7: 217 8: 257 9: 246 }

mapafrec(dic('x',pi,'y',_e,'z',pi))

Diccionario: { 2.718281828459045: 1 3.141592653589793: 2 }
