Función: Diccionario a XMLVolver

Descripción

Convierte un diccionario a representación de cadena XML

Los valores serán moldeados previamente por la biblioteca JSON-java

Cadena de entrada

xml

Cadena de salida

XML

Uso

xml(<diccionario>)

Ejemplos

xml(dic('a',12,'b','hi','c',verdadero,'d',100))

Texto: '<a>12.0</a><b>hi</b><c>true</c><d>100.0</d>'
