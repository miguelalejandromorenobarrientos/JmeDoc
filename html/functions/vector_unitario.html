Función: Vector UnitarioVolver

û = u|u|

Descripción

Crea un vector unitario bidimensional a partir de un argumento (en radianes) o normaliza un vector dado de cualquier dimensión

Cadena de entrada

unit

Cadena de salida

unit

Uso

unit(<terminal>) <terminal>::= <real> | <vector>

Ejemplos

unit(pi/4)

VectorEvaluado: [0.7071067811865476,0.7071067811865475]

unit(0)=v2i

Booleano: verdadero

unit(3,4)

VectorEvaluado: [0.6,0.8]

unit(0,1,10,5,-1)

VectorEvaluado: [0,0.08873565094161139,0.8873565094161139,0.44367825470805694,-0.08873565094161139]
