Función: Media AritméticaVolver

X̅= ∑Xi dim(X)

Descripción

Media aritmética de los elementos de un vector

Cadena de entrada

media

Cadena de salida

media

Uso

media(<vector>)

Ejemplos

media(2,4,12)

RealDoble: 6

media(rango(1,101))

RealDoble: 50.5

media(2,4-2ui)

Complejo: 3-I

la media de una matriz devuelve la media de las oolumnas:

media([1,2,3],[3,2,3])

VectorEvaluado: [2,2,3]

media de las filas trasponiendo:

media(trasp([1,2,3],[3,2,3]))

VectorEvaluado: [2,2.6666666666666665]
