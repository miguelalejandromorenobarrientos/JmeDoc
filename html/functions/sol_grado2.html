Función: Resolver Ecuación de Segundo GradoVolver

Descripción

Resuelve una ecuación de segundo grado ax2+bx+c=0 dados los coeficientes a, b y c y devuelve las dos soluciones como vector

Cadena de entrada

sol_grado2

Cadena de salida

sol_grado2

Uso

sol_grado2(<a>,<b>,<c>)

Parámetros
# Parámetro Descripción Valor por defecto
1 a Coeficiente ligado a la variable
2 b Coeficiente ligado a la variable
3 c Término independiente
Todos los coeficientes deben ser RealDoble o Complejo

Ejemplos

dos soluciones reales:

sol_grado2(1,-1,-1) x2-x-1=0

VectorEvaluado: [1.618033988749895,-0.6180339887498949] (proporción áurea ϕ)

dos soluciones complejas conjugadas:

sol_grado2(pi,_e,1) pi·x2+e·x+1=0

VectorEvaluado: [-0.43262798971613253+0.36213658845519686I,-0.43262798971613253-0.36213658845519686I]

una solución real doble:

sol_grado2(2,4,2) 2x2+4x+2=0

VectorEvaluado: [-1,-1]

eval(if(sol;#;1=sol;#;2,sol;#;1,sol),sol,sol_grado2(2,4,2)) 2x2+4x+2=0

RealDoble: -1

coeficientes complejos:

roundp(sol_grado2(î,-3-2î,-2+4î)) i·x2+(-3-2i)·x-2+4i=0

VectorEvaluado: [2-4I,I]

resolver lineal, resolver tercer grado, resolver sistema
