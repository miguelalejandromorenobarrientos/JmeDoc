Función: JuliaVolver

Descripción

Devuelve el nº de iteraciones, órbita y último valor del algoritmo de escape para el conjunto de Juliahttps://es.wikipedia.org/wiki/Conjunto_de_Julia. Si no escapa, el nº de iteraciones es Infinity.

Cadena de entrada

julia

Cadena de salida

julia

Uso

julia(<C>,<puntoZ>,<maxiter>[,<órbita>[,<valor_interior>]])

Parámetros
# Parámetro Descripción Valor por defecto
1 C Punto en el plano complejo que define al conjunto.
Para cada valor en el plano complejo existe un conjunto de Julia, que será conexo si este punto pertenece al conjunto de Mandelbrot
2 puntoZ Punto en el plano complejo
3 maxiter Máximo de iteraciones para considerar un punto como interior del fractal
4 órbita true para devolver órbita completa, false devuelve vector vacío false
5 valor_interior Valor devuelto para los puntos interiores del conjunto inf

Valor devuelto

Diccionario con las entradas

  • 'iter', nº de iteraciones (Infinity si el valor no escapa)
  • 'orbita', vector de valores de la órbita (vector vacío si <órbita> es false)
  • 'zn', último valor en la iteración n
  • 'z0', punto de evaluación inicial (=puntoZ)

Ejemplos

julia(cmplj(1,1),cmplj(0.1,0.2),10000)

Diccionario: { 'zn': 0.8592999999999998+3.0176I 'z0': 0.1+0.2I 'orbita': [] 'iter': 2 }
