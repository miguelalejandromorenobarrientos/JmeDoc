Función: Primo Aleatorio de N BitsVolver

Descripción

Devuelve un número primo entero pseudoaleatorio del número de bits especificado

Cadena de entrada

randprimo

Cadena de salida

randprimo

Uso

randprimo(<bits>[,<semilla>])

Parámetros
# Parámetro Descripción Valor por defecto
1 bits Longitud en bits del número
2 semilla
  • -1; siguiente nº aleatorio
  • 0; usar reloj del sistema
  • otro valor; valor fijo
 -1

Ejemplos

randprimo(100)

EnteroGrande: 869521766046576040388348839979

ordenar(conjunto(sucesion(randprimo(6),_,1,100)))

VectorEvaluado: [37,41,43,47,53,59,61]

[randprimo(4,pi),randprimo(4,pi)]

VectorEvaluado: [13,13]
