Función: AssertVolver

Descripción

Evalúa una condición, si es verdadera devuelve un valor, si es falsa se produce una excepción de aserción

Cadena de entrada

_assert_

Cadena de salida

_assert_

Uso

_assert_(<condición>,<valor>)

Parámetros
# Parámetro Descripción Valor por defecto
1 condición Condición para devolver el valor, o lanzar una excepción
2 valor Valor devuelto si la precondición se cumple

Ejemplos

_assert_(primo(7),7)

RealDoble: 7

_assert_(primo(8),8)

AssertException: <<<AssertException>>> en condicion "primo(8)"
