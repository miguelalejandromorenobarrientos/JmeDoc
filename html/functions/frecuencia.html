Función: FrecuenciaVolver

Descripción

Frecuencia de un elemento en un vector

Cadena de entrada

frec

Cadena de salida

frec

Uso

frec(<vector>,<elemento>)

Ejemplos

frec([4,3,2++,6,3,1,-1],3)

RealDoble: 3

frec([1,2,3,4],5)

RealDoble: 0

frec([[1,0],[2,1],[1,0],[2--,-1++],[-1,-1]],[1,0])

RealDoble: 3
