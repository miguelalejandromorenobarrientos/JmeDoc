Función: Media ponderadaVolver

X̅w= ∑Xi×wi ∑wi

Descripción

Devuelve la media ponderada de unos valores y sus pesos

Cadena de entrada

pond

Cadena de salida

pond

Uso

pond(<vectorValores>,<vectorPesos>)

Parámetros
# Parámetro Descripción Valor por defecto
1 vectorValores Vector de elementos a calcular media ponderada
2 vectorPesos Vector de pesos de la misma dimensión que el vector de valores, correspondiendo cada peso a cada valor

Ejemplos

pond([9,10,5,6],[1,2,2,3]) 9×1+10×2+5×2+6×38

RealDoble: 7.125

pond([3.12d,-1b,2+3ui],[4,5,1]) 4×3.12-1×5+(2+3I)×110

Complejo: 0.9480000000000001+0.3I

pond([[1,2],[3,4]],[2,3]) 2[1,2]+3[3,4]5

VectorEvaluado: [2.2,3.2]
