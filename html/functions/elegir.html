Función: ElegirVolver

Descripción

Selecciona uno o varios elementos de un vector, con o sin repetición, de manera aleatoria

Cadena de entrada

elegir

Cadena de salida

elegir

Uso

elegir(<vector>,<cantidad>[,<tipo>[,<semilla>]])

Parámetros
# Parámetro Descripción Valor por defecto
1 vector Vector de datos
2 cantidad Número de elementos a elegir. Para n=1 devolverá el elemento seleccionado, para un número mayor, un vector con todos los elementos seleccionados
3 tipo verdadero para seleccionar con repetición falso
4 semilla 0→reloj sistema, -1→siguiente valor, otro→valor fijo -1

Ejemplos

elegir([-1,3,ui,5],2)

VectorEvaluado: [5,I]

elegir(rango(10),5,verdadero)

VectorEvaluado: [9,0,3,9,0] con repetición

elegir([verdadero,falso],10,verdadero,1)

VectorEvaluado: [falso,verdadero,verdadero,verdadero,verdadero,verdadero,verdadero,falso,falso,falso]
