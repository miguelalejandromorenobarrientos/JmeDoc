Función: Convertir a RealDobleVolver

Descripción

Convierte un tipo de dato a RealDoble, excepto vectores:

  • RealDoble: el propio valor
  • Complejo: parte real del complejo (igual a función re)
  • RealGrande: convierte a RealDoble, con posible pérdida de precisión
  • EnteroGrande: convierte a RealDoble, con posible pérdida de precisión
  • Vectores: devuelve el vector con todos los elementos convertidos a RealDoble
  • Booleano: 0 para falso y 1 para verdadero
  • Texto: convierte a RealDoble a partir de una cadena representando un flotante válido

Cadena de entrada

realdoble

Cadena de salida

ℝ

Uso

realdoble(<elemento>)

Ejemplos

realdoble(rgpi)

RealDoble: 3.141592653589793

realdoble(11111111111111111111111b)

RealDoble: 1.1111111111111111E22

realdoble(2-3ui)

RealDoble: 2

realdoble(verdadero,falso)

VectorEvaluado: [1,0]

realdoble('1.23E-1')

RealDoble: 0.123

realdoble(tverdad(p->q,p,q))

VectorEvaluado: | 0 0 1 | | 0 1 1 | | 1 0 0 | | 1 1 1 |
