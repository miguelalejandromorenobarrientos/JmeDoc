Función: ERFCVolver

Descripción

Función error complementaria (ERFC)https://es.wikipedia.org/wiki/Funci%C3%B3n_error.

Con un solo parámetro, utiliza un algoritmo con un margen de error máximo de 1.2E-7, con dos, usa un método integral más preciso pero mucho más lento

Cadena de entrada

erfc

Cadena de salida

ERFC

Uso

erfc(<punto>[,<n>])

Parámetros
# Parámetro Descripción Valor por defecto
1 punto Punto de evaluación (real)
2 n Número de intervalos para aproximar el valor (n>0) mediante método integral. Si no se especifica, se usa el método polinomial ∅

Ejemplos

erfc(1)

RealDoble: 0.15729921223999332

erf(1,1e6)

RealDoble: 0.15729920705022837

Valor real: 0.15729920705028513
