Función: IndefinidoVolver

Descripción

Esta función permite detectar si una variable o una clave de diccionario está definida o devolver un valor opcional en caso contrario.
El principal propósito de esta función es su uso en JMEScript.

Cadena de entrada

indefinido

Cadena de salida

indefinido

Uso

indefinido(<varname>[,<valor_opcional>]) |indefinido(<diccionario>,<clave>[,<valor_opcional>])

Nota: en el caso de un parámetro, es necesario escribir la variable entre corchetes (ver ejemplos)

Parámetros
# Parámetro Descripción Valor por defecto
1 varname Nombre de la variable
1 diccionario
2|3 valor_opcional Valor a devolver si la variable o la clave está indefinida ∅
2 clave Clave del diccionario que puede o no estar definida

Valor devuelto

  • Si no se introduce el valor opcional, devuelve verdadero si la variable o clave están definidas, falso en caso contrario.
  • Si se introduce el valor opcional, en caso de que la variable o clave estén definidas, se devuelve su valor, en caso contrario el valor opcional.
