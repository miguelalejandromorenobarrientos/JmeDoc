Función: Para TodoVolver

Descripción

Devuelve verdadero si todos los elementos de un vector cumplen una condición

Cadena de entrada

paratodo

Cadena de salida

∀

Uso

paratodo(<item>,<vector>,<condición{item}>)

Parámetros
# Parámetro Descripción Valor por defecto
1 item variable con cada elemento del vector
2 vector Vector de datos
3 condición Condición que debe cumplir todo elemento

Ejemplos

paratodo(x,[2,3,5,7],primo(x))

Booleano: verdadero

paratodo(x,[2,3,1,5,7],primo(x))

Booleano: falso
