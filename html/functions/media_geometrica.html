Función: Media GeométricaVolver

GM(X)= (∏Xi) dim(X)-1

Descripción

Media geométrica de los elementos de un vector

Cadena de entrada

mediag

Cadena de salida

GM

Uso

mediag(<vector>)

Ejemplos

mediag(10,1000)

RealDoble: 100

mediag(rango(1,101))

RealDoble: 37.62310047409742

mediag(2,4-2ui)

Complejo: 2.9106933804507102-0.687121499445025I
