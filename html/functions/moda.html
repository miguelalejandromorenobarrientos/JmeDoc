Función: ModaVolver

Descripción

Moda muestral de los elementos de un vector.

El resultado será un vector con la moda de una serie de datos unimodal o las modas de una serie de datos multimodal

Cadena de entrada

moda

Cadena de salida

moda

Uso

moda(<vector>)

Ejemplos

moda(2,6,-1,-4,0,5,6,2,2,1)

VectorEvaluado: [2]

moda(4,1,1,4,5)

VectorEvaluado: [4,1]

moda(falso,verdadero,falso,falso,verdadero)

VectorEvaluado: [falso]
