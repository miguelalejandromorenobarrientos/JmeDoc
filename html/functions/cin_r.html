Función: Cinemática: Posición FinalVolver

r(t) = r0 + v0t + a2t2

Descripción

Cálculo escalar y vectorial de la posición final a partir de posición inicial, velocidad inicial, aceleración y tiempo.
Esta ecuación es una de las BIG 4 que determinan el movimiento uniformemente acelerado.

Cadena de entrada

cin_r

Cadena de salida

cin_r

Uso

cin_r(<r0>,<v0>,<a>,<t>)

Parámetros
# Parámetro Descripción Valor por defecto
1 r0 Posición inicial como escalar o vector
2 v0 Velocidad inicial como escalar o vector
3 a Aceleración como escalar o vector
4 t Tiempo como flotante

Ejemplos

cin_r(6,2,4,5)

RealDoble: 66

cin_r([2,3],[3,2],[1,1],10)

VectorEvaluado: [82,73]

cin_r([-9,2,3],[1,2,3],[-1,-2,-3],-3)

VectorEvaluado: [-16.5,-13,-19.5]
