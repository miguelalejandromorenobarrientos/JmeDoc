Función: Dimensión de MatrizVolver

Descripción

Dimensión de una matriz. La dimensión se obtiene como vector 2D [filas,columnas]

Cadena de entrada

matdim

Cadena de salida

matdim

Uso

matdim(<matriz>)

Ejemplos

matdim([1,2,3],[4,5,6])

VectorEvaluado: [2,3]

matdim(1,2,3,4) Para vectores las dimensión se devuelve como vector columna

VectorEvaluado: [4,1]

matdim([[]])

VectorEvaluado: [1,0]

matdim([])

VectorEvaluado: [0,1]
