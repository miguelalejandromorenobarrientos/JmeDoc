Función: Redondeo Entero PróximoVolver

Descripción

Redondea números muy cercanos a un entero (por defecto menos de 1E-12) a ese entero

Cadena de entrada

roundp

Cadena de salida

roundp

Uso

roundp(<número>)

Ejemplos

sqr(sqrt(8))

RealDoble: 8.000000000000002

roundp(sqr(sqrt(8)))

RealDoble: 8

cos(pi/2)

RealDoble: 6.123233995736766E-17

roundp(cos(pi/2))

RealDoble: 0

(3-2î)^5

Complejo: -596.9999999999999-122.00000000000018I

roundp((3-2î)^5)

Complejo: -597-122I

mrot([0,0,1],pi/2)

VectorEvaluado: | 6.123233995736766E-17 -1 0 | | 1 6.123233995736766E-17 0 | | 0 0 1 |

roundp(mrot([0,0,1],pi/2))

VectorEvaluado: | 0 -1 0 | | 1 0 0 | | 0 0 1 |

roundp(dic(next(2),next(2))) (en diccionarios se aplica la función al valor de cada entrada)

Diccionario: {2.0000000000000004=2}
