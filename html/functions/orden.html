Función: OrdenVolver

Descripción

Devuelve el orden de magnitud de un número

Cadena de entrada

ord

Cadena de salida

ord

Uso

ord(<numero>)

Ejemplos

ord(0)

RealDoble: 0

ord(9)

RealDoble: 0

ord(prev(10))

RealDoble: 0

ord(10)

RealDoble: 1

ord(100)

RealDoble: 2

ord(5000)

RealDoble: 3

ord(0.05)

RealDoble: -2

ord(100000b!)

RealDoble: 456573

ord(3e6,1.7e-1000d)

VectorEvaluado: [6,-1000]
