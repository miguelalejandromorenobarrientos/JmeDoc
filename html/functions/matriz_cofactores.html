Función: Matriz de CofactoresVolver

Descripción

Matriz de cofactores de una matriz (cuadrada)

Cadena de entrada

mcof

Cadena de salida

mcof

Uso

mcof(<matriz>)

Ejemplos

mcof([1,2],[3,4])

VectorEvaluado: | 4 -3 | | -2 1 |

mcof([ui,2+3ui,0],[-ui,0,3],[1,2,-2ui])

VectorEvaluado: | -6 5 -2I | | -6+4I 2 2+I | | 6+9I -3I -3+2I |
