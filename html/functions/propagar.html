Función: PropagarVolver

Descripción

  • Propaga una expresión para cada elemento de un vector (o fila de matriz), creando un vector con los elementos mapeados
  • Propaga una expresión para cada valor de un diccionario, creando un diccionario con las mismas claves y los valores mapeados

Cadena de entrada

propagar

Cadena de salida

propagar

Uso

propagar(<función{varname}>,<varname>,<vector>|<diccionario>)

Parámetros
# Parámetro Descripción Valor por defecto
1 función{varname} Expresión que depende de una variable de iteración sobre el vector o diccionario
2 varname Variable de iteración de la que depende la función que itera sobre el vector o los valores del diccionario
3 vector|diccionario Vector de valores o diccionario a mapear por la función

Ejemplos

propagar(sqr(item),item,rango(11))

VectorEvaluado: [0,1,4,9,16,25,36,49,64,81,100]

propagar(suma(fila),fila,[[1,2,5d,ui],[0,-1,4b,1-ui],[0,1,3,-inf]])

VectorEvaluado: [8+I,4-I,-Infinity]

propagar(round(item,3),item,dic('x',phi,'y',_e,'z',pi))

Diccionario: {'x'=1.618, 'y'=2.718, 'z'=3.142}
