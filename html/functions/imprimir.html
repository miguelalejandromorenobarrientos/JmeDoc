Función: ImprimirVolver

Descripción

Imprimir la cadena de salida de un terminal en la salida por defecto del sistema.
Nota para desarrolladores: los streams de salida se pueden modificar mediante Imprimir#setOut, Imprimir#setErr

Cadena de entrada

imprimir

Cadena de salida

imprimir

Uso

imprimir(<dato>[,<salida>[,<no_comillas>[,<eol>]]])

Parámetros
# Parámetro Descripción Valor por defecto
1 dato Dato a imprimir en consola
2 salida true para salida por stream 'out', false 'err' true
3 no_comillas (solo Texto); true para no poner comillas en cadenas de texto false
4 eol true para salto de línea true

Valor devuelto

El valor devuelto es la propia salida del terminal como texto

Ejemplos

imprimir('Hola Mundo!!!')

(out): 'Hola Mundo!!!'

Texto: 'Hola Mundo!!!'

imprimir('Hola Mundo!!!',falso)

(err): 'Hola Mundo!!!'

Texto: 'Hola Mundo!!!'

imprimir('Hola Mundo!!!',verdadero,verdadero)

(out): Hola Mundo!!!

Texto: 'Hola Mundo!!!'

[imprimir('Hola Mundo!!!',verdadero,verdadero,falso),imprimir('Adios Mundo!!!',verdadero,verdadero)]

(out): Hola Mundo!!!Adios Mundo!!!

VectorEvaluado: ['Hola Mundo!!!','Adios Mundo!!!']
