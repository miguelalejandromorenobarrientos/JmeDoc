Función: Ordenar vectorVolver

Descripción

Ordena un vector de números reales

Cadena de entrada

ordenar

Cadena de salida

ordenar

Uso

ordenar(<vectorReal>[,<sentido>])

Parámetros
# Parámetro Descripción Valor por defecto
1 vector Vector de números reales
2 sentido Booleano verdadero para orden decreciente, y falso para creciente falso (creciente)

ordenar(sucesion(rand(-10,10),_,1,10))
