Función: BisiestoVolver

Descripción

Comprueba si un determinado año es bisiesto, o cual es el año bisiesto previo/posterior

Cadena de entrada

bisiesto

Cadena de salida

bisiesto

Uso

bisiesto(<año>[,<dirección>])

Parámetros
# Parámetro Descripción Valor por defecto
1 año Año a comprobar si es bisiesto. Si se introduce un booleano (no importa cual), se usará el año del sistema
2 dirección Constante _prev_ para año bisiesto anterior al dado y _sig_ para el siguiente ∅

Ejemplos

bisiesto(2000)

Booleano: verdadero

bisiesto(2100)

Booleano: falso

bisiesto(-100)

Booleano: falso

bisiesto(1980)

Booleano: verdadero

bisiesto(verdadero)

Booleano: falso (2018)

bisiesto(2018,_sig_)

RealDoble: 2020

bisiesto(2097,_sig_)

RealDoble: 2104

bisiesto(3,_ant_)

RealDoble: 0

bisiesto(2018,_ant_)

RealDoble: 2016
