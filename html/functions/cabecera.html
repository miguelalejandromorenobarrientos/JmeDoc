Función: CabeceraVolver

Descripción

Devuelve el primer elemento de un vector (o primera fila de una matriz).

Nota: esta función es equivalente al operador elemento para el índice igual a 1, pero es algo más eficiente

Cadena de entrada

cabecera

Cadena de salida

cabecera

Uso

cabecera(<vector>)

Ejemplos

cabecera(3,6,2)

RealDoble: 3

cabecera(falso,[1,2],4)

Booleano: falso

cabecera(mat1(4))

VectorEvaluado: [1,0,0,0]

cabecera([])

FuncionException: <<<FuncionException>>> en funcion "cabecera": Vector nulo --> cabecera(Vector:[])
