Función: Constante JavaVolver

Descripción

Devuelve una constante pública estática Java convertida a JME con las mismas conversiones que el valor devuelto por la función java.

El uso de esta función requiere de algún conocimiento de Java

Cadena de entrada

jconst

Cadena de salida

JCONST

Uso

jconst(<constname>)

Ejemplos

jconst('java.math.RoundingMode.UP')

Texto: 'UP'

jconst('java.lang.Math.PI','java.lang.Math.E')

VectorEvaluado: [3.141592653589793,2.718281828459045]
