Función: Multiplicar VectorVolver

Descripción

Multiplica los elementos de un vector

Cadena de entrada

mul

Cadena de salida

mul

Uso

suma(<vector>)

Ejemplos

mul(2,5,-1,7)

RealDoble: -70

mul(rango(1,101)) (=100!)

RealDoble: 9.33262154439441E157

mul(rango(11,21)) (=20!10!)

RealDoble: 6.704425728E11

mul(3.2d,5,4b,-1+2î)

Complejo: -64+128I

mul(3,[2,3],5,-1,[0,4],[1,3])

VectorEvaluado: [-180,-540]

mul(3,[0,4],[1,3],[2,3],5,-1)

VectorEvaluado: [-360,-540] (el producto escalar no es asociativo)

mul([[1,2],[3,4]],mat1(2),[[0,0],[0,0]])

VectorEvaluado: | 0 0 | | 0 0 |

mul([])

FuncionException: <<<FuncionException>>> en funcion "mul": Vector vacio --> mul(VectorEvaluado:[])
