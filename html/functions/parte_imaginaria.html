Función: Parte ImaginariaVolver

Descripción

Parte imaginaria de un complejo (para no complejos, devuelve 0)

Cadena de entrada

im

Cadena de salida

im

Uso

im(<complejo>)

Ejemplos

im(4+2î)

RealDoble: 2

im(ui)

RealDoble: 1

im(polar(3,pi/4))

RealDoble: 2.1213203435596424

im([3+2ui,cmplj(-1,-1),î],[pi,3.2d,123b])

VectorEvaluado: | 2 -1 1 | | 0 0 0 |
