Función: Convertir a BooleanoVolver

Descripción

Convierte un tipo de dato a booleano, excepto vectores:

  • Números: si son iguales a 0, falso, en otro caso verdadero
  • Vectores: devuelve el vector con todos los elementos convertidos a booleano
  • Booleano: el mismo booleano
  • Texto: permite introducir un valor booleano a partir de un conjunto de keywords mayor (ver último ejemplo)

Cadena de entrada

booleano

Cadena de salida

ℬ

Uso

booleano(<elemento>)

Ejemplos

booleano(3)

Booleano: verdadero

booleano(0)

Booleano: falso

booleano(î)

Booleano: verdadero

booleano(0d)

Booleano: falso

booleano(falso)

Booleano: falso

booleano(0+0î,verdadero,5,[-1,0])

VectorEvaluado: [falso,verdadero,verdadero,[verdadero,falso]]

booleano(falso)

Booleano: falso

cadenas de texto válidas como verdadero:

and(booleano('verdadero','true','verdad','cierto','si','yes','v','t','1'))

Booleano: verdadero

cadenas de texto válidas como falso:

or(booleano('falso','false','mentira','no','f','0'))

Booleano: falso
