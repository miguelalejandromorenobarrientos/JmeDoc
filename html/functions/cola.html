Función: ColaVolver

Descripción

Devuelve el último elemento de un vector (o última fila de una matriz).

Nota: esta función es equivalente al operador elemento para el índice igual a -1, pero es algo más eficiente

Cadena de entrada

cola

Cadena de salida

cola

Uso

cola(<vector>)

Ejemplos

cola(3,6,2)

RealDoble: 2

cola(falso,[1,2],4)

RealDoble: 4

cola(mat1(4))

VectorEvaluado: [0,0,0,1]

cola([])
