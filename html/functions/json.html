Función: Diccionario a JSONVolver

Descripción

Convierte un diccionario a representación de cadena JSON

Los valores serán moldeados previamente por la biblioteca JSON-java

Cadena de entrada

json

Cadena de salida

JSON

Uso

json(<diccionario>)

Ejemplos

json(dic('a',12,'b','hi','c',verdadero,100,[2,3]))

Texto: '{"a":12,"b":"hi","100.0":[2,3],"c":true}'
