Función: Media Cuadrática (RMS)Volver

RMS(X) = √ ∑Xi2 dim(X)

Descripción

Media cuadrática, valor cuadrático medio o RMS de los elementos de un vector

Cadena de entrada

rms

Cadena de salida

RMS

Uso

rms(<vector>)

Ejemplos

rms(1,2,3)

RealDoble: 2.160246899469287

rms(rango(1,101))

RealDoble: 58.16786054171152

rms(2,4-2ui)

Complejo: 3.1075479480600743-1.2871885058111652I
