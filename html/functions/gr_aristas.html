Función: Lista de Aristas de un GrafoVolver

Descripción

Devuelve un vector con las aristas del grafo.

Cadena de entrada

gr_aristas

Cadena de salida

GRAFO.aristas

Uso

gr_aristas(<grafo>)

Valor devuelto

El valor devuelto es una matriz de dimensión |A|x2 conteniendo las parejas de nodos de cada arista [[v1i,v1f],[v2i,v2f],…], dados por su índice

Ejemplos

Digrafo ponderado (aristas representadas por valor distinto de infinito):

gr_aristas(gr_nuevo([[inf,inf,100],[-1,inf,-1],[inf,inf,inf]],true,true))

VectorEvaluado: | 0 2 | | 1 0 | | 1 2 |
