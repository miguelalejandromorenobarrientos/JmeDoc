Función: ContarVolver

Descripción

Cuenta los elementos de un vector que cumplen una condición

Cadena de entrada

contar

Cadena de salida

contar

Uso

contar(<condición{varItem}>,<varItem>,<vector>)

Parámetros
# Parámetro Descripción Valor por defecto
1 condición{varItem} Condición que deben cumplir los elementos que pasen el corte
2 varItem Variable que contiene a cada elemento del vector en cada iteración
3 vector Vector de datos

Ejemplos

contar(item<0,item,[1,-2,5,0,pi,-10,1,20])

RealDoble: 2

contar(primo(item),item,rango(1,101))

RealDoble: 25
