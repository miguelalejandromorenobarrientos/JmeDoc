Función: Parte RealVolver

Descripción

Parte real de un complejo (para no complejos, devuelve el propio número)

Cadena de entrada

re

Cadena de salida

re

Uso

re(<complejo>)

Ejemplos

re(4+2î)

RealDoble: 4

re(ui)

RealDoble: 0

re(polar(3,pi/4))

RealDoble: 2.121320343559643

re([3+2ui,cmplj(-1,-1),î],[pi,3.2d,123b])

VectorEvaluado: | 3 -1 0 | | 3.141592653589793 3.2 123 |
