Función: Resolver Ecuación de Tercer GradoVolver

Descripción

Resuelve una ecuación de tercer grado ax3+bx2+cx+d=0 dados los coeficientes a, b, c y d y devuelve las tres soluciones como vector.

Cadena de entrada

sol_grado3

Cadena de salida

sol_grado3

Uso

sol_grado3(<a>,<b>,<c>,<d>)

Parámetros
# Parámetro Descripción Valor por defecto
1 a Coeficiente ligado a la variable
2 b Coeficiente ligado a la variable
3 c Coeficiente ligado a la variable
4 d Término independiente
Todos los coeficientes deben ser reales

Ejemplos

tres soluciones reales:

sol_grado3(1,-6,11,-6) x3-6x2+11x-6=0

VectorEvaluado: [3,1,2]

una solución real (primera siempre) y dos soluciones complejas conjugadas:

sol_grado3(-3,-3,-1,7) -3x3-3x2-x+7=0

VectorEvaluado: [1,-1+1.1547005383792515I,-1-1.1547005383792515I]
