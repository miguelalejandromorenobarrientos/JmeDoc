Función: InversoVolver

Descripción

Inverso de un número o matriz inversa de una matriz regular

Cadena de entrada

inv

Cadena de salida

inv

Uso

inv(<elemento>) <elemento>::= <número>|<matrizRegular>

Ejemplos

inv(3)

RealDoble: 0.3333333333333333

inv(3d)

RealGrande: 0.3333333333333333333333333333333333

inv(0)

RealDoble: Infinity

inv(inf)

RealDoble: 0

inv(ui)

Complejo: -I

inv(2-2ui)

Complejo: 0.25+0.25I

inv([1,ui],[3,2+ui])

VectorEvaluado: | 0.25+0.75I 0.25-0.25I | | -0.75-0.75I 0.25+0.25I |
