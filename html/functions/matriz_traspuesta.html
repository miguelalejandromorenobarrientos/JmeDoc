Función: Matriz TraspuestaVolver

Descripción

Traspuesta de una matriz

Cadena de entrada

trasp

Cadena de salida

trasp

Uso

trasp(<matriz>)

Ejemplos

trasp([1,2,3,4,5],[-1,0,0,0,1])

VectorEvaluado: | 1 -1 | | 2 0 | | 3 0 | | 4 0 | | 5 1 |

trasp([1,2],[3,4])

VectorEvaluado: | 1 3 | | 2 4 |

matriz fila a matriz columna:

trasp(1,2,3,4,5)

VectorEvaluado: | 1 | | 2 | | 3 | | 4 | | 5 |

trasp(rango(3))

VectorEvaluado: | 0 | | 1 | | 2 |
