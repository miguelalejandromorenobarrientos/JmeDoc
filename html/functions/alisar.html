Función: AlisarVolver

Descripción

Redondea números muy cercanos a cero (por defecto menos de 1E-12)

Cadena de entrada

alisar

Cadena de salida

alisar

Uso

alisar(<número>)

Ejemplos

cos(pi/2)

RealDoble: 6.123233995736766E-17

alisar(cos(pi/2))

RealDoble: 0

(-1)^0.5

Complejo: 6.123233995736766E-17+I

roundp((-1)^0.5)

Complejo: I

mrot([0,0,1],pi/2)

VectorEvaluado: | 6.123233995736766E-17 -1 0 | | 1 6.123233995736766E-17 0 | | 0 0 1 |

alisar(mrot([0,0,1],pi/2))

VectorEvaluado: | 0 -1 0 | | 1 0 0 | | 0 0 1 |
