Función: ExisteVolver

Descripción

Devuelve verdadero si algún elemento de un vector cumple una condición

Cadena de entrada

existe

Cadena de salida

∃

Uso

existe(<item>,<vector>,<condición{item}>)

Parámetros
# Parámetro Descripción Valor por defecto
1 item variable con cada elemento del vector
2 vector Vector de datos
3 condición Condición que debe cumplir al menos un elemento

Ejemplos

existe(x,[1,2,4,6],primo(x))

Booleano: verdadero

existe(x,[1,4,6],primo(x))

Booleano: falso
