Función: Crear DiccionarioVolver

Descripción

Crea un nuevo diccionario a partir de un vector de 2n valores (clave/valor), o una cadena JSON o XML.

Tanto los valores como las claves pueden ser cualquier tipo de terminal, aunque cuando se utiliza una cadena JSON/XML, los valores serán moldeados previamente por la biblioteca JSON-java

Cadena de entrada

dic

Cadena de salida

DIC

Uso

dic(<vector>|<texto_JSON>|<texto_XML>|<complejo>) <texto_XML>::= xml: <contenido_XML>

Ejemplos

dic('nombre','Viviana','edad',38)

Diccionario: {'nombre'='Viviana', 'edad'=38}

dic('{nombre:Viviana,edad:38}')

Diccionario: {'nombre'='Viviana', 'edad'=38}

dic('xml:<nombre>Viviana</nombre><edad>38</edad>')

Diccionario: {'nombre'='Viviana', 'edad'=38}

dic(-2-î)

Diccionario: {'RE'=-2, 'IM'=-1}
