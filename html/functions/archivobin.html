Función: Cargar Archivo BinarioVolver

Descripción

Carga dato (terminal) desde un archivo binario.

Para usar esta función debe estar activado el acceso a archivos en JME.

Para Desarrolladores

Puede activarse/desactivarse la carga de archivos desde JME mediante el método CargarArchivoBin#setActivado.

Cadena de entrada

archivobin

Cadena de salida

archivobin

Uso

archivobin(<ruta_archivo>)

Ejemplos

archivobin('c:/<a_path>/test.bin')

Diccionario: { 'txt': 'hola mundo' 'n': 100 }
