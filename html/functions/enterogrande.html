Función: Convertir a EnteroGrandeVolver

Descripción

Convierte un tipo de dato a EnteroGrande, excepto vectores:

  • RealDoble: valor del RealDoble como EnteroGrande con pérdida de precisión si tenía decimales
  • Complejo: parte real del complejo como EnteroGrande con pérdida de precisión si tenía decimales
  • RealGrande: convierte a EnteroGrande con pérdida de precisión si tenía decimales
  • EnteroGrande: devuelve el pripio valor
  • Vectores: devuelve el vector con todos los elementos convertidos a EnteroGrande
  • Booleano: 0 para falso y 1 para verdadero
  • Texto: convierte a EnteroGrande a partir de una cadena representando un entero válido en formato decimal, hexadecimal (0x|#) u octal (0)

Cadena de entrada

enterogrande

Cadena de salida

ℤb

Uso

enterogrande(<elemento>)

Ejemplos

enterogrande(pi)

EnteroGrande: 3

enterogrande(123456789123456789.5d)

EnteroGrande: 123456789123456789

enterogrande(3.5-2ui)

EnteroGrande: 3

enterogrande(verdadero,falso)

VectorEvaluado: [1,0]

enterogrande(20'125')

EnteroGrande: 125125125125125125125125125125125125125125125125125125125125

enterogrande('#FF') Hexadecimal

EnteroGrande: 255

enterogrande('-#FF') Hexadecimal

EnteroGrande: -255

enterogrande('0x7FFFFFFFFFFFFFFF') Hexadecimal (máximo valor aceptado)

EnteroGrande: 9223372036854775807

enterogrande('-0x8000000000000000') Hexadecimal (mínimo valor aceptado)

EnteroGrande: -9223372036854775808

enterogrande('0777777777777777777777') Octal (máximo valor aceptado)

EnteroGrande: 9223372036854775807

enterogrande('-01000000000000000000000') Octal (mínimo valor aceptado)

EnteroGrande: -9223372036854775808

enterogrande(tverdad(!p,p))

VectorEvaluado: | 0 1 | | 1 0 |
