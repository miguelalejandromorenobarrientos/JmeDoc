Función: Matriz de TraslaciónVolver

Descripción

Matriz de traslación en coordenadas homogéneas a partir de un vector de desplazamientos

Cadena de entrada

mtras

Cadena de salida

mtras

Uso

mtras(<desplazamiento>) <desplazamiento>::= <vectorNum> vector de desplazamiento en cada eje

Ejemplos

mtras(6,4,-2)

VectorEvaluado: | 1 0 0 6 | | 0 1 0 4 | | 0 0 1 -2 | | 0 0 0 1 |

mtras(4,rgpi,-2,0,rgphi,rge,0,0)
