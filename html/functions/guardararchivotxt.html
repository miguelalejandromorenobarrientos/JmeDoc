Función: Guardar Archivo de TextoVolver

Descripción

Guarda la salida de un terminal en un archivo de texto o añade por el final (append).

Para usar esta función debe estar activado el acceso a archivos en JME.

Para Desarrolladores

Puede activarse/desactivarse el guardado de archivos desde JME mediante el método GuardarArchivoTxt#setActivado.

Cadena de entrada

guardararchivotxt

Cadena de salida

guardararchivotxt

Uso

guardararchivotxt(<ruta_archivo>,<dato>[,<append>])

Parámetros
# Parámetro Descripción Valor por defecto
1 ruta_archivo Ruta y nombre para localizar archivo
2 dato Terminal a guardar como cadena de texto de salida
3 append verdadero para añadir al final en lugar de reemplazar todo el archivo (si ya existía) falso

Ejemplos

guardararchivotxt('c:/<a_path>/hiworld.txt',[5,4d,falso,[]])

hiworld.txt: [5,4,falso,[]]

guardararchivotxt('c:/<a_path>/hiworld.txt','Hi World',verdadero)

hiworld.txt: [5,4,falso,[]]Hi World
