Función: EnumerarVolver

Descripción

  • Enumera los elementos de un vector mediante un contador con inicio y paso, o usando los elementos de otro vector índices. Este vector debe tener al menos la misma dimensión
  • Convierte diccionario en vector enumerado por las claves

Cadena de entrada

enumerar

Cadena de salida

enumerar

Uso

enumerar(<vector>,(<inicio>[,<paso>]) | <vector_índices>) | enumerar(<diccionario>)

Parámetros
# Parámetro Descripción Valor por defecto
1 vector|diccionario Vector de elementos a enumerar o diccionario
2 inicio Valor del primer índice (se convertirá a RealGrande para mayor precisión) ∅
2 vector de índices Vector de índices que se asignarán consecutivamente a los del vector de elementos. Debe contener suficientes índices ∅
3 paso Incremento del índice (se convertirá a RealGrande para mayor precisión) 1|∅

Ejemplos

enumerar(sucesion(sqr(n),n,1,10),1)

VectorEvaluado: | 1 1 | | 2 4 | | 3 9 | | 4 16 | | 5 25 | | 6 36 | | 7 49 | | 8 64 | | 9 81 | | 10 100 |

enumerar([14,5,-4,-13,-22],-10,-5)

VectorEvaluado: | -10 14 | | -15 5 | | -20 -4 | | -25 -13 | | -30 -22 |

enumerar(rango(10),0,0.001)

VectorEvaluado: | 0 0 | | 0.001 1 | | 0.002 2 | | 0.003 3 | | 0.004 4 | | 0.005 5 | | 0.006 6 | | 0.007 7 | | 0.008 8 | | 0.009 9 |

enumerar([phi,_e,pi],partir('# áureo, # de Euler, pi',', '))

VectorEvaluado: | '# áureo' 1.618033988749895 | | '# de Euler' 2.718281828459045 | | 'pi' 3.141592653589793 |

enumerar(dic('{x:2,y:9,z:-1}'))

VectorEvaluado: | 'x' 2 | | 'y' 9 | | 'z' -1 |
