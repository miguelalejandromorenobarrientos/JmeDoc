Operador: CocienteVolver

Tipo y precedencia

BINARIO (50)

Descripción

Cociente (división) entre numerador y denominador…

  • … escalares reales/complejos
  • … vector/matriz y escalares reales/complejos
  • … vectores
  • … matrices de MxN/NxN

En el caso de números, el resultado se ajustará al número más preciso, excepto para números complejos, en que el resultado será complejo.

Cadena de entrada

/

Cadena de salida

/

Uso

<numerador>/<denominador>

Ejemplos

3/5

RealDoble: 0.6

-2b/4

RealGrande: -0.5

3/4e10d

RealGrande: 7.5E-11

î/(2-3î)

Complejo: -0.23076923076923078+0.15384615384615385I

[9,3]/3

VectorEvaluado: [3,1]

[[1,2,3],[4,5,6]]/ui

VectorEvaluado: | -I -2I -3I | | -4I -5I -6I |

1/[3,4] (vector inverso respecto al producto escalar 1/v=v/|v|2)

VectorEvaluado: [0.12,0.16]

[3,4]/[3,4] (vector por su inverso = 1)

RealDoble: 1

1/[[2,3],[4,5]] (matriz inversa, puede usarse también inv([2,3],[4,5]))

VectorEvaluado: | -2.5 1.5 | | 2 -1 |

[[1,2],[3,4],[5,6]]/[[1,2],[3,4]] (matriz numerador por inversa del denominador)

VectorEvaluado: | 1 0 | | 0 1 | | -1 2 |
